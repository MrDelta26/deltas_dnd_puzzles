# Delta's DnD Puzzles
A collection of puzzles I (Delta26) will use for my DnD campaigns.

Note To PLAYERS:
Some of the files on this repository may be used as clues. If you have found this repository, maybe either A: Check with your DM to see what you should and should not view. Or, B: Don't spoil stuff for yourself.

Note to DUNGEON MASTERS:
Most pages in this are intended to be accessed by either a QR code or other similar methods. There will eventually be a page with hints and solutions you can tell your players. Be patient, please.

Note to ALL:
This is very early in development. Some puzzles may be unfinished, and a lot has not even been started. Don't expect much right away.
